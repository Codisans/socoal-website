# Socoal Website

Built using [Tailwind CSS](https://tailwindcss.com) and [Next.js](https://nextjs.org), based on the Radiant template from [Tailwind Plus](https://tailwindcss.com/plus).

## Getting started

To get started with this template, first install the npm dependencies:

```bash
npm install
```

Next, run the development server:

```bash
npm run dev
```

Finally, open [http://localhost:3000](http://localhost:3000) in your browser to view the website.

## SEO Configuration

This website includes comprehensive SEO setup. See the following files for details:

- **`QUICK-START-SEO.md`** - Quick reference for deployment
- **`SEO-SETUP-COMPLETE.md`** - Complete setup summary
- **`SEO.md`** - Comprehensive SEO guide
- **`SEO-CHECKLIST.md`** - Step-by-step checklist
- **`STRUCTURED-DATA-EXAMPLES.md`** - Code examples

### SEO Features Included

- ✅ XML Sitemap (`/sitemap.xml`)
- ✅ Robots.txt (`/robots.txt`)
- ✅ PWA Manifest (`/manifest.json`)
- ✅ Open Graph & Twitter Cards
- ✅ Structured Data support
- ✅ Security headers
- ✅ Image optimization

**Important:** Before deploying, verify the domain in `src/app/sitemap.ts`, `src/app/robots.ts`, `src/app/layout.tsx`, and `src/components/structured-data.tsx`.
